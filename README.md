# 地址处理脚本

这个脚本用于处理 Excel 文件中的地址信息，并在输出文件中提取省、市、姓名、电话和详细地址。

## 环境要求

- Python 3.x
- pandas 库

## 安装

1. 克隆或下载此代码仓库到本地计算机。
2. 在命令行中切换到项目目录。
3. 运行以下命令来安装所需的依赖库：

   ```shell
   pip install pandas #当没有安装pandas时
   python3 main.py

##  说明
1、运行前您需要准备一个Excel文档，并在第一个工作表的第一列添加需要提取的信息
2、将Excel文档放在运行目录下
3、运行main.py即可

当然，您也可以把operationtext.py导入到您的项目中，并执行getuserinfo()函数，结果返回一个字典类型数据，其中包含{name,phone_number,address,pro,city} #其中name为称呼，phone_number为号码，address为地址，pro为省份，city为城市
